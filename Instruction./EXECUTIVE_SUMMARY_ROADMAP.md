## 🎨 Visual Brand Reference

### Color System
```
Primary:   #7F77DD (Purple) - Main brand color
Secondary: #1D9E75 (Teal)   - Innovation, growth
Accent:    #D85A30 (Coral)  - Energy, action
Neutral:   #000000 / #FFFFFF / #999999 (Gray)

Usage:
- Primary: Buttons, links, highlights
- Secondary: Cards, badges, accents
- Accent: CTAs, important elements
- Neutral: Typography, backgrounds
```

### Typography
```
Headlines: Instrument Sans / Space Grotesk (geometric, modern)
Body:      Inter / Anthropic Sans (clean, readable)
Code:      JetBrains Mono (monospace)

Sizes:
H1: 48-56px  (landing, page titles)
H2: 32-40px  (section titles)
H3: 24-28px  (subsection)
Body: 16px   (paragraph text)
```
